## 📖 Overview

Learnify AI is a revolutionary learning platform that combines artificial intelligence with blockchain technology. Built on the Internet Computer Protocol (ICP), it offers personalized quizzes, AI tutoring, gamified learning experiences, and blockchain-verified achievements.

### ✨ Key Features

- 🤖 **AI-Powered Quizzes** - Adaptive quizzes generated by AI based on difficulty and topic
- 🎓 **AI Tutor Assistant** - Real-time help and explanations from an intelligent tutor
- 🏆 **Blockchain Badges** - Earn NFT credentials stored on the blockchain
- 📊 **Progress Analytics** - Visual dashboards with pie charts and performance tracking
- 🎯 **Personalized Learning** - Custom difficulty levels and topic selection
- 🌐 **Decentralized** - Fully hosted on the Internet Computer blockchain

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript, SCSS, Chart.js, EmailJS
- **Backend**: Motoko (Internet Computer)
- **AI Integration**: Perplexity API via Node.js bridge server
- **Blockchain**: Internet Computer Protocol (ICP)
- **Authentication**: Internet Identity
- **Build Tool**: Vite
- **Package Manager**: npm

## 🚀 Quick Start

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [DFX SDK](https://internetcomputer.org/docs/current/developer-docs/setup/install) (v0.15.0 or higher)
- [Git](https://git-scm.com/)

### 1. Clone the Repository

```
git clone https://github.com/yourusername/learnify-ai.git
cd learnify-ai
```

### 2. Install Dependencies

```
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the root directory:

```
PERPLEXITY_API_KEY=your_perplexity_api_key_here
CANISTER_ID_LEARNIFY_AI_BACKEND=your_backend_canister_id
CANISTER_ID_LEARNIFY_AI_FRONTEND=your_frontend_canister_id
CANISTER_ID_INTERNET_IDENTITY=your_ii_canister_id
DFX_NETWORK=local
```

### 4. Start the Bridge Server

The AI features require a bridge server to connect to the Perplexity API:

```
# In a separate terminal
cd bridge-server
node server.js
```

The server will run on `http://localhost:4000`

### 5. Deploy Locally

```
# Start the local Internet Computer replica
dfx start --clean --background

# Deploy the canisters
dfx deploy

# Start the frontend development server
npm start
```

### 6. Access the Application

- **Frontend**: `http://localhost:4943/?canisterId=`
- **Backend Candid UI**: `http://localhost:4943/?canisterId=`

## 🔧 Configuration

### EmailJS Setup (for Feedback)

1. Create an account at [EmailJS](https://www.emailjs.com/)
2. Set up a service and template
3. Update the service ID and template ID in `main.js`:

```
const result = await emailjs.sendForm(
    'your_service_id',    // Replace with your service ID
    'your_template_id',   // Replace with your template ID
    '#feedbackForm'
);
```

### Perplexity API Setup

1. Get an API key from [Perplexity](https://www.perplexity.ai/)
2. Add it to your `.env` file
3. The bridge server handles all AI requests

## 🐛 Troubleshooting

### Common Issues

**1. "Chart.js not available" error**
- Ensure Chart.js CDN is loaded in HTML
- Check browser console for loading errors

**2. "EmailJS is not defined" error**
- Verify EmailJS script is loaded
- Check your service ID and template ID

**3. Backend connection failed**
- Ensure DFX is running (`dfx start`)
- Check canister IDs in environment variables

**4. AI features not working**
- Start the bridge server (`node server.js`)
- Verify Perplexity API key is set
