## 📖 Overview

Learnify AI is a revolutionary learning platform that combines artificial intelligence with blockchain technology. Built on the Internet Computer Protocol (ICP), it offers personalized quizzes, AI tutoring, gamified learning experiences, and blockchain-verified achievements.

---

## ✨ Key Features

* 🤖 **AI-Powered Quizzes** - Adaptive quizzes generated by AI based on difficulty and topic
* 🎓 **AI Tutor Assistant** - Real-time help and explanations from an intelligent tutor
* 🏆 **Blockchain Badges** - Earn NFT credentials stored on the blockchain
* 📊 **Progress Analytics** - Visual dashboards with charts and performance tracking
* 🎯 **Personalized Learning** - Custom difficulty levels and topic selection
* 🌐 **Decentralized** - Fully hosted on the Internet Computer blockchain

## 🚀 Quick Start

### Prerequisites

* [Node.js](https://nodejs.org/) (v16 or higher)
* [DFX SDK](https://internetcomputer.org/docs/current/developer-docs/setup/install)
* [Git](https://git-scm.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/learnify-ai.git
cd learnify-ai
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file:

```env
PERPLEXITY_API_KEY=your_perplexity_api_key_here
CANISTER_ID_LEARNIFY_AI_BACKEND=your_backend_canister_id
CANISTER_ID_LEARNIFY_AI_FRONTEND=your_frontend_canister_id
CANISTER_ID_INTERNET_IDENTITY=your_ii_canister_id
DFX_NETWORK=local
```

### 4. Start the Bridge Server

```bash
cd bridge-server
node server.js
```

### 5. Deploy Locally

```bash
dfx start --clean --background
dfx deploy
npm start
```

### 6. Open in Browser

* Frontend: `http://localhost:4943/?canisterId=...`
* Candid UI: `http://localhost:4943/?canisterId=...`
